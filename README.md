### Notes on customization

- **Theme**: Currently set to `tokyonight`. Other options: `radical`, `dracula`, `gruvbox`, `ocean`, `dark`. Just change the parameter in the stat URLs.
- **Hide private repos**: Set `hide_private=false` if you want to show private repo count.
- **Profile banner**: You can add a custom banner image at the top with: `<img src="https://raw.githubusercontent.com/Maddyrampant/Maddyrampant/master/banner.png" />`
- **Custom README stats**: Visit [github-readme-stats](https://github.com/anuraghazra/github-readme-stats) for more widget types (e.g., streak counter, WakaTime, repositories).
- **Dynamic badges**: The shields.io badges automatically update when you click them for stats like forks or stars.
- **Stargazer animation**: There are popular animated GIFs at the very top (e.g., `waving-hand`) you can toggle on/off by adding/removing `?use_emoji=true` and setting `style="display:none"` — though some prefer the cleaner look without it.
